Privacy Policy

Last updated: March 28, 2026

The Golden Rule: Product Party runs 100% locally inside your browser. Your scraped data, store credentials, and extraction output are never sent to, stored on, or shared with any external servers, analytics providers, or third-party databases.

1. What This Extension Does

Product Party: Shopify Product Scraper & CSV Extractor is a Chrome Extension built on Manifest V3. It extracts publicly available product data from Shopify-powered online stores and converts it into downloadable CSV or JSON files on your local machine.

2. Data Collection

We do not collect any personal data. Product Party does not:

  • Transmit any data to remote servers
  • Use cookies or tracking pixels
  • Store browsing history
  • Collect email addresses, names, or any personal identifiers
  • Use analytics, telemetry, or crash reporting services
  • Communicate with any external API beyond the Shopify store you are actively visiting

3. Data Processing

All data extraction and file generation occurs entirely within your browser using Chrome's native APIs. Specifically:

  • Product data is fetched directly from Shopify's public /products.json endpoints on the store you are visiting
  • CSV/JSON conversion is performed locally using JavaScript inside the extension's Service Worker
  • Image ZIP files are generated locally using an embedded copy of JSZip — no external CDN is loaded
  • Brand Kit assets (logos, favicons) are fetched directly from the store's own CDN and packaged locally

4. Permissions & Justifications

This extension requests the following Chrome permissions:

Permission Justification
activeTab Required to execute extraction scripts only when the user explicitly clicks the extension popup on the active tab.
scripting Required to inject the DOM scanner into the active tab to identify product handles, theme names, and brand asset URLs.
downloads Required to save the generated CSV, JSON, and ZIP files directly to the user's local filesystem.
storage Required to persist the user's export preferences (column exclusions, filename conventions, HTML stripping) across sessions using Chrome Sync Storage.
<all_urls> (Optional) An optional, user-toggled permission that enables the "Auto-Detect Shopify" background listener. When enabled, the extension badge lights up when a Shopify store is detected. This permission is not granted by default.

5. User Settings Storage

Product Party uses chrome.storage.sync to save your export preferences. This data is stored in your Chrome profile and synced across your signed-in browsers via Google's native sync infrastructure. The only data stored is:

  • Whether to strip HTML tags from descriptions (boolean)
  • Whether to exclude the Body_HTML column (boolean)
  • Filename prefix style preference (string: "store" or "date")

No personal information is ever stored.

6. Third-Party Services

Product Party does not integrate with any third-party services, analytics platforms, advertising networks, or data brokers. The only external HTTP requests made are to the Shopify store that the user is actively visiting, using Shopify's publicly accessible JSON endpoints.

7. Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect any information from children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last updated" date at the top of this page. Continued use of the extension after changes constitutes acceptance of the updated policy.
